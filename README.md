# Repository

This repository will have the main structure in `src/streamlit_app.py`

To run this streamlit project: 
$ cd "/path-to-the-project-root/Team-Root-"
$ python3 -m venv .venv
$ source .venv/bin/activate


#### verify
$ which python
$ python -V 
$ python -m pip install --upgrade pip 
$ pip install streamlit
$ python3 -m pip install pandas numpy joblib plotly


#### run
$ streamlit run src/streamlit_app.py
